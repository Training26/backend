# blog-post

## Project setup

```
npm install
```

### Run The Server

```
node index.js
```

### Routes Docs

### Users Route

#### Get Users

localhost:5000/api/v1/users

#### Get User

localhost:5000/api/v1/users/id

#### Delete post

localhost:5000/api/v1/users/id

### Post Route

#### Get Posts

localhost:5000/api/v1/posts

#### Get post

localhost:5000/api/v1/post/id

#### Delete post

localhost:5000/api/v1/post/id

### Messages Route

#### Get messages

localhost:5000/api/v1/messages

#### Get message

localhost:5000/api/v1/message/id

#### Delete message

localhost:5000/api/v1/message/id
